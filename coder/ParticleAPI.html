The technical documentation of the Lua Scripting API for the Particle System Plugin. Contains details about plugin data entities (purpose, property field names, expected value types), built-in functions (purpose, arguments, returned values) and built-in constants.

Conventions used in this document

This document follows certain guidelines for maintaining consistency in presenting information clearly.

Plugin data entity descriptions

Plugin data entities are shortly characterized (i.e. what data they represent inside the plugin) and all of their field names are listed in a table, each with a description of what data / value type they hold what is the expected value range (if user assigns a value). There is also a specification of whether the field is read-only, meaning that the plugin allows the user to get the value of a field, but does not allow directly assigning new values it (which doesnâ€™t imply that the value given by said field cannot be changed through other means).
