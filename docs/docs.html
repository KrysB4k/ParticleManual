bindFunction (id, func)
Binds the function func to index ID. This function can then be invoked via Flipeffect 1 from Plugin_ParticleSystem. Can only be called in module-loading phase.

Parameters:

  • id int the ID to bind to; constrained to [1, 1024]
  • func func the function to bind
bindGroup (id, func)
Binds the group init function to index ID. This init function then be invoked via Flipeffect 1 from Plugin_ParticleSystem. Can only be called in module-loading phase.

Parameters:

  • id int the ID to bind to; constrained to [1, 1024]
  • func func the ParticleGroup of which init will bind

createGroup (init, update)
Creates a new ParticleGroup object with a given init and update function. Can only be called in the module-loading phase.

Parameters:

  • init optional func the initialization function; pass nil if no init function desired
  • update optional func the update function; pass nil if no update function desired

Returns:

    ParticleGroup the ParticleGroup object
